# 浮光 macOS

浮光是一款 macOS 效率工具，提供截图、长截图、录屏、贴图、OCR、剪贴板、图片处理、Markdown 等常用功能。

## 下载

请前往 [Releases](https://github.com/jackotom/mac-app/releases) 下载最新版安装包。

当前版本：`4.0.1`

系统要求：macOS 14 或更高版本，支持 Apple 芯片和 Intel 芯片。

## 安装

1. 下载 `FuGuang-4.0.1.dmg`。
2. 打开安装包。
3. 将“浮光”拖入“应用程序”文件夹。
